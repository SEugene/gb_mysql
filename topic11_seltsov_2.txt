Задание 2.1. В базе данных Redis подберите коллекцию для подсчета посещений с определенных IP-адресов.
Решается с помощью создания hash-таблицы: ключ - IP address, значение - количество посещений. Для записи данных используется команда HSET, для извлечения - HGET

Задание 2.2. При помощи базы данных Redis решите задачу поиска имени пользователя по электронному адресу и наоборот, поиск электронного адреса пользователя по его имени.
Решаем с помощью двух hash-таблиц.
Создаем таблицы:
HSET user_mail user_name email
HSET mail_user email user_name
Выполняем поиск:
HGET user_mail "user_name" - ищем почту
HGET mail_user "email" ищем пользователя
